# How to Run This Project

## Quick Start

1. Download the dataset from Kaggle
2. Install dependencies
3. Run the analysis notebook

## Step-by-Step Guide

### 1. Download Dataset

Go to: https://www.kaggle.com/datasets/fedesoriano/stroke-prediction-dataset

Download and extract `healthcare-dataset-stroke-data.csv` to the `data/` folder.

### 2. Install Dependencies

```bash
cd stroke-prediction-analysis
pip install -r requirements.txt
```

### 3. Run Analysis

```bash
jupyter notebook notebooks/01_stroke_prediction_analysis.ipynb
```
